# Running the program

## Create a virtual environment

```python3 -m venv .venv```

## Activate the virtual environment
```. .venv/bin/activate```

## Install Flask
```pip install Flask```

## Install Requests
```pip install requests```

## Run the flask application
```python3 -m flask --app src/app run``` 

# Build / Rebuild Database

## Run db.py
```python3 src/db.py run```
